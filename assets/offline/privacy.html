Privacy Policy

Kami menghormati privasi Anda. Halaman ini menjelaskan bagaimana data ditangani dalam template ini.

📅 Terakhir diperbarui: 13 Juni 2024
🔒 Ringkasan: Flutter WebView Template adalah template open-source yang tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi pengguna akhir. Template ini hanya menjadi "jembatan" antara aplikasi Flutter dan website yang Anda konfigurasikan.
1. Tentang Template Ini

Flutter WebView Template ("Template") adalah kode sumber open-source yang dikembangkan oleh weruka.dev dan didistribusikan di bawah lisensi MIT. Template ini bukan aplikasi akhir — melainkan kerangka dasar yang digunakan pengembang untuk membangun aplikasi mereka sendiri.

Kebijakan privasi ini berlaku untuk halaman landing ini (index.html) dan halaman pendukungnya. Kebijakan privasi aplikasi yang dibangun menggunakan template ini sepenuhnya menjadi tanggung jawab pengembang masing-masing.

2. Data yang Tidak Kami Kumpulkan

Template ini tidak mengumpulkan data berikut:

  • Informasi identitas pribadi (nama, email, nomor telepon)
  • Data lokasi atau GPS
  • Riwayat penjelajahan atau aktivitas dalam aplikasi
  • Data kamera, mikrofon, atau file pengguna
  • Informasi perangkat atau identifiers unik
  • Data analitik atau telemetri penggunaan
3. Halaman Offline & Assets Lokal

Halaman ini (index.html) berjalan sepenuhnya secara lokal di perangkat pengguna sebagai fallback saat tidak ada koneksi internet. Halaman ini:

  • Tidak mengirimkan data ke server eksternal
  • Tidak menggunakan cookie pihak ketiga
  • Tidak memuat skrip analitik (Google Analytics, Facebook Pixel, dll.)
  • Menggunakan localStorage dan sessionStorage hanya untuk keperluan demonstrasi fitur WebView (bagian Test Fitur) — data ini tidak dikirim ke mana pun
4. Izin Perangkat

Aplikasi yang dibangun menggunakan template ini mungkin meminta izin perangkat berikut. Semua izin bersifat opsional dan dapat dinonaktifkan lewat app_config.dart:

  • Kamera & Mikrofon: Diperlukan untuk fitur WebRTC (video call di browser). Data tidak disimpan oleh template.
  • Lokasi: Diperlukan untuk navigator.geolocation. Data tidak dikirim ke weruka.dev.
  • Storage/File: Diperlukan untuk upload dan download file. File diproses secara lokal.

Data yang diperoleh dari izin-izin tersebut diteruskan langsung ke website yang dikonfigurasi oleh pengembang aplikasi — bukan ke weruka.dev.

5. Tautan Eksternal

Halaman ini mengandung tautan ke situs eksternal seperti GitHub, WhatsApp, Telegram, Twitter, Facebook, dan LinkedIn untuk keperluan berbagi. Kami tidak bertanggung jawab atas kebijakan privasi situs-situs tersebut. Kunjungi situs masing-masing untuk informasi lebih lanjut.

6. Tanggung Jawab Pengembang Aplikasi

Jika Anda menggunakan template ini untuk membangun aplikasi, Anda bertanggung jawab untuk:

  • Menyusun kebijakan privasi untuk aplikasi Anda sendiri
  • Mematuhi regulasi privasi yang berlaku (GDPR, UU PDP Indonesia, dll.)
  • Menginformasikan pengguna tentang data yang dikumpulkan oleh website yang dimuat di WebView
  • Mengelola izin perangkat sesuai kebutuhan aplikasi Anda
7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan akan ditandai dengan memperbarui tanggal "Terakhir diperbarui" di bagian atas halaman ini. Versi terbaru selalu tersedia di repository GitHub.
